# Tia's Portfolio

## Installation
Git clone repo
Run `npm i` || `npm install` in terminal
Run `live server` in terminal


## Style Guide

* Font-family: 'Vidaloka', serif;
* Primary color: #EDBEAD;
* Secondary color: #A23E48;
* Tertiary color: #F7F3EB;
